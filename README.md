# Масажи по домовете

Статичен сайт за masajipodomovete.com – професионален масаж у дома. София и област.

## Стек

- Next.js 13 (App Router)
- Tailwind CSS
- TypeScript

## Локално

```bash
npm install
npm run dev
```

Отвори [http://localhost:3000](http://localhost:3000).

## Deploy (Vercel)

1. Пушни кода в GitHub репото.
2. В Vercel: Import репото, задай env vars от `env.example` (и от Supabase ако ползваш).
3. След всеки push има автоматичен deploy.

Подробни стъпки за GitHub, Vercel и Supabase: виж **SETUP_MASAJIPODOMOVETE.md** в проекта rabotim-com.
